Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability. It is widely used for web development, data science, automation, artificial intelligence, and more. Python's syntax is easy to learn, making it a great choice for beginners.

print("Hello World")

Variables

A Python variable is a name that stores a value, acting as a container for data. Variables allow you to store, modify, and reuse data in your program. Variable names folllow certain rules:

  • Must Start with a Letter or Underscore (_)
  • Can Only Contain Letters, Digits, and Underscores
  • Case-Sensitive
  • Cannot Use Python Keywords (Reserved Words)
  • No Special Characters Except (_)

Declaring Variables

There are multiple ways to decleare a variable in Python. Bellow are a few of those ways.

Declaration with a type:

int x = 5

Declaration without a type:

x = 5

Assigning multiple variables with different values:

x, y, z = 1, 2, 3

Assigning multiple variables the same value:

x = y = z = 5

If Else Statements

An if-else statement in Python is a conditional statement used to execute different blocks of code based on whether a condition is true or false. An if-else statemaent looks as follows:

if condition:
statement 1
else:
statement 2

If there are multiple coditions you can use elif:

if condition 1:
statement 1
elif condition 2:
statement 2
elif condition 3:
statement 3

While Statements

A while statement in Python is a type of loop that executes a block of code repeatedly as long as a given condition remains true. Bellow is the structure of a while statement:

while condition:
statement
